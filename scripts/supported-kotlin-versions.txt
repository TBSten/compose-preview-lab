# SSOT for the Kotlin versions that compose-preview-lab supports.
# One version per line (blank lines and lines starting with `#` are comments).
# CI workflows and compiler-plugin-test scripts read this file.
#
# Supported range: Kotlin 2.3.21 and later.
#   Older versions (2.1.20–2.3.20) are intentionally not supported any more —
#   `collectAllModulePreviews()` requires the FIR top-level declaration generator
#   pipeline that only stabilized on KLIB targets in 2.3.21 (KT-82395 fix).
#   The compat-k210 / compat-k222 / compat-k2220 / compat-k230 modules still ship
#   in the published plugin jar so existing builds with single-module
#   `collectModulePreviews()` keep loading, but they are no longer covered by CI.
#
# To add a new version:
#   1. Append a line below.
#   2. Verify locally with `./scripts/compiler-plugin-test.sh <new-version>`.
#   3. If API drift requires it, add a new compat module
#      (compiler-plugin/compat-k* / settings.gradle.kts / META-INF/services).
