# Compose Preview Lab について

## 概要

**Compose Preview Lab** は、Kotlin Multiplatform/Jetpack Composeプロジェクトの開発者向けに設計された、インタラクティブなコンポーネント・プレイグラウンド作成ライブラリです。
従来の `@Preview` アノテーションを強化し、パラメータ操作やイベント可視化を可能にすることで、単なる静的スナップショットを超えた「動く」UIカタログを実現します。

## 強み・利用するメリット

### 1. 静的プレビューを超えたインタラクティブな検証

- フィールド操作（Field API）をサポート  
  `fieldValue { ***Field(defaultValue) }` により、Preview画面内で値を自由に操作可能。
- イベント発火の可視化  
  `onEvent()` を呼ぶだけで、例えばボタンクリックなどの挙動も確認できます。

### 2. 準備が圧倒的に簡単

- 既存の `@Preview` を `PreviewLab { ... }` で囲むだけ。  
- 別の Story 用ソースセットへの移行等は不要。既存資産がそのまま活用可能です。

### 3. Compose Multiplatform 対応

- Android, JVM, JS, Wasm, iOS（KMP）に対応。
- web 上でもカタログをそのまま共有できるため、デザイナーや非エンジニアともリアルタイムに確認可能。

### 4. 独自型パラメータやカスタマイズ性

- Fieldを自作でき、業務ごとの独自入力UIや、運用に即したカスタム挙動も追加可能。
- SelectableField などの便利なユーティリティも標準搭載。

### 5. Storytaleとの違い

- 既存Previewをそのまま活用可能（Storytaleは専用セットへの移動が必須）
- 将来、ソースコード表示やビジュアルリグレッションテストにも発展予定

## 想定される利用シーン

- Pull Requestレビュー時の手動テスト効率化
- 新人・他チームへのコンポーネント仕様共有
- デザイナー・非エンジニアとのUI仕様すり合わせ
- ドキュメントとしてのUIカタログ生成
