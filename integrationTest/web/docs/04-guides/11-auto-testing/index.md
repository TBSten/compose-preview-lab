# Auto Testing

当初 Compose Preview Lab は手動テストをより効率的に行うために設計されていましたが、
`PreviewLabState` などの API を利用することで、**自動テスト** にも活用できます。

:::info ここで扱うテストの種類
- 単体テスト（Compose UI テスト + PreviewLabState）  
- 将来的な拡張の方向性（テストコード自動生成 / Property based testing / VRT）  
:::
