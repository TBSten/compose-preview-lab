# Extensions

Compose Preview Lab は、外部ライブラリの型に対応した Field を提供する **Extension モジュール** を用意しています。

これらのモジュールは `starter` には含まれていないため、必要に応じて個別に依存関係を追加してください。

## 利用可能な Extension

| Extension | 対応ライブラリ | 説明 |
|-----------|---------------|------|
| `extension-kotlinx-datetime` | [kotlinx-datetime](https://github.com/Kotlin/kotlinx-datetime) | LocalDate, LocalTime, TimeZone などの Field を提供 |
